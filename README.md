🪐 3-Body Simulation (Sun–Earth–Jupiter)

This project simulates a three-body gravitational system including Sun, Earth, and Jupiter.

🚀 Features
Multi-body gravitational interaction
Realistic orbital perturbations
Energy conservation tracking
Animated orbital motion with trails
🧠 Physics Concepts
Three-body problem
Gravitational perturbation
Numerical integration (Verlet method)
🛠️ Tech Stack
Python
NumPy
Matplotlib
▶️ How to Run
pip install -r requirements.txt
python main.py
📊 Output
Orbital interaction between Earth and Jupiter
Perturbed trajectories
Energy stability visualization
